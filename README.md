# ✈️ TurTakip — Turizm Yönetim Sistemi

Ekip bazlı, tarayıcıda çalışan turizm takip sistemi. Takvim üzerinden turistleri, turları, uçuşları ve ödemeleri yönetin.

## 🚀 Hızlı Başlangıç

### Yerel Çalıştırma
`index.html` dosyasını doğrudan tarayıcıda açın.

## 📋 Özellikler

- 📅 **Yıllık Takvim** — 12 aylık görünüm, etkinlik olan günler turuncu
- 🗓️ **Aylık Takvim** — Semboller ile tur/uçuş/transfer gösterimi
- 👁️ **Günlük Görünüm** — O günkü tüm turistler ve aktiviteleri
- 👤 **Turist Profili** — Tüm bilgilere sekmeli erişim
- ✈️ **Uçuş Takibi** — Giriş/çıkış uçuşları
- 🚌 **Transfer Takibi** — Havalimanı ↔ Otel
- 💳 **Ödeme Takibi** — EUR/USD/TRY, ödeme durumu
- 🗺️ **Tanımlı Turlar** — Önceden tanımlı tur listesi
- 💾 **JSON Dışa/İçe Aktarım** — Veri yedekleme

## 🌐 GitHub Pages'te Ücretsiz Yayınlama

1. [github.com](https://github.com)'da ücretsiz hesap açın
2. Yeni repo oluşturun → `turizm-takip`
3. Tüm dosyaları repoya yükleyin
4. Settings → Pages → Deploy from branch → `main` → Save
5. Adresiniz: `kullaniciadiniz.github.io/turizm-takip`

## 🔧 Teknoloji

- Saf HTML + CSS + JavaScript (bağımlılık yok)
- Veriler: Browser LocalStorage
- Hosting: GitHub Pages (ücretsiz)
